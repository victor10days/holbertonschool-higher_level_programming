# JavaScript - DOM Manipulation

This project contains JavaScript exercises focusing on DOM manipulation, event handling, and API requests.

## Learning Objectives

- How to select HTML elements in JavaScript
- Differences between ID, class and tag name selectors
- How to modify HTML element styles and content
- How to modify the DOM
- How to make requests with Fetch API
- How to listen/bind to DOM and user events

## Files

- `0-script.js` - Updates header text color to red
- `1-script.js` - Changes header color when clicking on element
- `2-script.js` - Adds red class to header on click
- `3-script.js` - Toggles header class between red and green
- `4-script.js` - Adds list items dynamically
- `5-script.js` - Updates header text content
- `6-script.js` - Fetches Star Wars character data
- `7-script.js` - Fetches and displays Star Wars movies
- `8-script.js` - Fetches translation data
